📊 Call Center Performance Dashboard

This project presents an interactive Call Center Performance Dashboard built using Microsoft Power BI to analyze operational efficiency, agent performance, and customer satisfaction metrics. The dashboard helps stakeholders quickly monitor key performance indicators and identify opportunities to improve service quality.

🎯 Project Objective

The goal of this project is to transform raw call center data into meaningful insights that help businesses:

Monitor service level performance

Track agent productivity

Analyze customer satisfaction trends

Improve response and resolution times

🗂 Dataset Overview

The dataset contains call center operational data including:

Agent ID

Channel (Call / Chat / Email)

First Response Time

Handle Time

Call Status (Answered / Abandoned)

CSAT Score

Date and Time of Interaction

Data was processed and analyzed using SQL and data transformation techniques before visualization.

📈 Dashboard Features

The dashboard provides several key insights:

Total Calls Handled – Overall call volume in the system

Answer Rate – Percentage of answered calls vs total calls

Average Handle Time (AHT) – Average time spent resolving customer queries

First Response Time Analysis – Response efficiency tracking

Customer Satisfaction (CSAT) – Overall customer feedback score

Channel-wise Performance – Comparison between communication channels

Agent Performance Analysis – Identify top-performing agents

🛠 Tools & Technologies

Data Visualization: Microsoft Power BI

Data Analysis: SQL

Data Processing: Microsoft Excel

📊 Key Insights

Identified agents with the highest CSAT scores and fastest response times

Analyzed service level agreement (SLA) performance by channel

Highlighted peak call handling periods to support workforce planning

💡 Business Impact

This dashboard enables managers and operations teams to:

Make data-driven decisions

Optimize workforce management

Improve customer experience
